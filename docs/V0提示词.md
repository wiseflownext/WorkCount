# V0 提示词 - 工计宝原型图

将以下提示词分段输入V0，每段生成一组页面。

---

## 提示词1：登录页 + 员工首页

```
Design a mobile app UI prototype for a Chinese manufacturing worker payroll app called "工计宝". 
Target users are 40-60 year old factory workers with low tech literacy. 

Requirements:
- Mobile only (Android), Chinese language only
- Extra large fonts (body ≥ 18sp, buttons ≥ 20sp), high contrast
- Large touch targets (min 48x48dp), simple flat layout
- Max 2 levels of navigation depth
- Color scheme: Blue primary (#1976D2) for actions, Green (#4CAF50) for earnings/money, Red (#F44336) for alerts

Page 1 - Login Page:
- Phone number input (large, numeric keyboard)
- Password input (with show/hide toggle)
- Large "登录" (Login) button, full width
- Clean, minimal, no registration (admin creates accounts)
- App logo and name "工计宝" at top

Page 2 - Worker Home Page:
- Top: Greeting "你好, [姓名]" + today's date
- Middle: Today's earnings card showing "今日收入: ¥XXX.XX" in large green text
- Two large square buttons side by side:
  - "计件录入" (Piece Entry) with a counting icon
  - "计时打卡" (Time Clock) with a clock icon
- Bottom section: "今日记录" list showing today's entries (type, product, amount)
- Bottom tab bar: 首页(Home) / 日报(Daily Report) / 工资条(Payslip) / 我的(Profile)
- If there's an active time tracking session, show a prominent green pulsing banner at top: "正在计时中... [duration]"
```

## 提示词2：计件录入 + 计时打卡

```
Continue the "工计宝" manufacturing payroll app. Same design system: extra large Chinese fonts, high contrast, big buttons, for 40-60 year old factory workers.

Page 3 - Piece Entry Page (计件录入):
- Title bar: "计件录入"
- Step 1: Product/Process selector - large scrollable list of items showing [编号] [名称] [单价¥X.XX/件]
  - Each item is a large tappable card (full width, min 60dp height)
  - Show search bar at top (optional)
  - Selected item highlighted in blue
- Step 2: After selecting product, show:
  - Product name and current unit price displayed prominently
  - Large number input for quantity (with +/- stepper buttons, big numeric display)
  - Auto-calculated amount shown: "合计: ¥XXX.XX" in large green text
- Large "提交" (Submit) button at bottom, full width
- After submit: success toast with amount, return to home

Page 4 - Time Clock Page (计时打卡):
- Title bar: "计时打卡"
- State A (Not clocked in):
  - Work content input: text field or dropdown to select/type work description
  - Optional: link to a product/process from the list
  - Giant green circular "开始工作" (Start Work) button in center (at least 120dp diameter)
- State B (Clocked in / Working):
  - Show work content at top
  - Large timer display in center showing elapsed time "HH:MM:SS" 
  - Pulsing green border/glow to indicate active
  - Below timer: "已工作 X 小时 X 分钟" in plain text
  - Giant red circular "结束工作" (End Work) button
- State C (After ending):
  - Summary card: work content, start time, end time, duration (rounded to half hours), rate, amount earned
  - "确认" button to save, "继续工作" to start new session
```

## 提示词3：员工日报 + 工资条

```
Continue the "工计宝" app. Same design rules: extra large Chinese text, high contrast, big touch targets.

Page 5 - Worker Daily Report (我的日报):
- Title: "我的日报"
- Date picker at top (default today, swipe left/right to change date)
- Summary card at top:
  - "计件收入: ¥XX.XX" 
  - "计时收入: ¥XX.XX"
  - "日合计: ¥XX.XX" (large, green, bold)
- Below: chronological list of all records for that day
  - Each record card shows:
    - Type badge: "计件" (blue) or "计时" (orange)
    - Product/work content name
    - Details: quantity×price or duration×rate
    - Amount: ¥XX.XX (right aligned)
- Empty state: friendly message "今日暂无记录" with illustration

Page 6 - Worker Payslip (我的工资条):
- Title: "我的工资条"  
- Toggle tabs: "周报" / "月报"
- Week view:
  - Week selector (上一周 / 本周 / 下一周)
  - Bar chart or simple table showing daily earnings Mon-Sun
  - Summary: "本周合计: ¥XXXX.XX"
- Month view:
  - Month selector
  - Calendar heatmap or daily list showing each day's total
  - Summary card:
    - "计件总额: ¥XXXX.XX"
    - "计时总额: ¥XXXX.XX" 
    - "本月合计: ¥XXXX.XX" (extra large, green)
  - Breakdown table: date | piece income | time income | daily total
```

## 提示词4：管理员首页 + 员工管理

```
Continue "工计宝" app. Now design the ADMIN pages. Admin uses same app but with management capabilities. Same design system but can be slightly more information-dense than worker pages.

Page 7 - Admin Home (管理员首页):
- Top: "管理员面板" + today's date
- Summary dashboard cards:
  - "今日在岗人数: XX人"
  - "今日总产出: ¥XXXXX.XX"
  - "正在计时: X人"
- Quick action buttons row:
  - "员工管理" / "产品管理" / "数据报表" / "数据导出"
- Below: Today's activity feed - recent records from all workers
  - Each entry: [员工名] [计件/计时] [产品] [金额] [时间]
- Bottom tab: 首页 / 报表 / 导出 / 设置

Page 8 - Employee Management (员工管理):
- Title: "员工管理"
- "添加员工" button at top right
- Employee list, each card shows:
  - Name, phone number
  - Time rate: "时薪: ¥XX/半小时"
  - Status badge: "在职"(green) / "离职"(gray)
  - Tap to edit
- Add/Edit Employee form:
  - 姓名 (Name input)
  - 手机号 (Phone input, numeric)
  - 初始密码 (Password input, only on creation)
  - 时薪 (Hourly rate input, ¥ per half hour)
  - 状态 toggle (在职/离职)
  - Save / Cancel buttons
```

## 提示词5：产品管理 + 数据报表 + 导出

```
Continue "工计宝" admin pages. Same design system.

Page 9 - Product/Process Management (产品工序管理):
- Title: "产品工序管理"
- Filter tabs: "全部" / "产品" / "工序"
- "添加" button at top right
- List of products/processes, each card:
  - [编号] [名称]
  - "单价: ¥X.XX/[单位]"
  - Type badge: "产品"(blue) / "工序"(purple) 
  - Status: "启用"(green) / "停用"(gray)
- Add/Edit form:
  - 编号 (Code), 名称 (Name)
  - 类型 dropdown (产品/工序)
  - 单价 (Unit price input)
  - 单位 dropdown (件/个/组/套)
  - 备注 (Remark, optional)
  - 状态 toggle
- Price change: when editing price, show warning "单价变更仅对新记录生效"

Page 10 - Admin Reports (数据报表):
- Title: "数据报表"
- Toggle: "日报" / "周报" / "月报"
- Employee filter dropdown (全部员工 / specific employee)
- Date/Week/Month selector
- Summary cards: total piece income, total time income, grand total
- Data table below:
  - Daily view: employee | piece count | piece income | hours | time income | total
  - Week/Month view: date | total piece | total time | daily total
  - Totals row at bottom
- Scrollable table for many employees

Page 11 - Data Export (数据导出):
- Title: "数据导出"
- Date range picker (start date - end date)
- Employee selector (全部 / select specific)
- Data type checkboxes: "计件明细" / "计时明细" / "薪资汇总"
- Preview summary: "将导出 XX 条记录"
- Large "导出Excel" button
- Export history list below showing previous exports
```
